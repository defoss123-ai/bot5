# MEXC Martingale GUI

## Важно

Реальная торговля связана с рисками и может привести к потере средств. Используйте
реальный режим только при полном понимании работы стратегии и рисков. Рекомендуется
сначала протестировать логику в Paper mode и убедиться, что параметры настроены корректно.

## Запуск

```bash
pip install -r requirements.txt
python main.py
```

## Реальный режим

1. Убедитесь, что заполнены API ключ и секрет (для Spot).
2. Включите чекбокс **"Реальный режим"**.
3. При необходимости оставьте включёнными защиты:
   - **Чистый старт (отменить ордера перед запуском)** — удаляет открытые ордера по паре.
   - **Отменять ордера при стопе** — отменяет ордера при нажатии "Стоп".
   - **Авто-перезапуск** — автоматически начинает новый цикл после TP (по умолчанию выключен).
4. Нажмите **"Старт (Реальный)"**.

## Параметры страховочных ордеров (Paper)

- **Кол-во страховочных ордеров**: допустимый диапазон 0..50.
- **Шаг страховочного ордера, %**: допустимый диапазон 0.1..50.
- **Сумма первого ордера (USDT)**: допустимый диапазон 1..100000.
- **Сумма страховочного ордера (USDT)**: допустимый диапазон 1..100000.

## Параметры тейк-профитов (Paper)

- **TP1/TP2/TP3 %**: допустимый диапазон 0.1..50.
- **Доля TP1/TP2/TP3 %**: допустимый диапазон 0..100, сумма долей должна быть 100.
